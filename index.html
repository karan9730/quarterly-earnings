Code: Quick NPV/IRR Utilities

import numpy as np

cash_flows = [-1000, 250, 300, 400, 500]
r = 0.10  # discount rate

# Net Present Value
npv = sum(cf / ((1 + r) ** t) for t, cf in enumerate(cash_flows))
print(f"NPV @ 10% = {npv:.2f}")

# Internal Rate of Return (naïve)
irr = np.irr(cash_flows)  # numpy_financial.irr in newer stacks
print(f"IRR = {irr:.2%}")

Financial Formulae

Net Present Value:

$$\mathrm{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t}$$

Compound Annual Growth Rate (CAGR):

$$\mathrm{CAGR} = \left(\frac{\text{Ending}}{\text{Beginning}}\right)^{\tfrac{1}{n}} - 1$$

Sharpe Ratio:

$$\mathrm{Sharpe} = \frac{E[R_p - R_f]}{\sigma_p}$$

Use these when explaining valuation, growth, and risk-adjusted returns.
